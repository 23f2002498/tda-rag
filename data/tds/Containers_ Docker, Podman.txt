<article class="markdown-section" id="main"><h2 id="containers-docker-podman"><a class="anchor" data-id="containers-docker-podman" href="#/docker?id=containers-docker-podman"><span>Containers: Docker, Podman</span></a></h2><p><a href="https://www.docker.com/" rel="noopener" target="_blank">Docker</a> and <a href="https://podman.io/" rel="noopener" target="_blank">Podman</a> are containerization tools that package your application and its dependencies into a standardized unit for software development and deployment.</p><p>Docker is the industry standard. Podman is compatible with Docker and has better security (and a slightly more open license). In this course, we recommend Podman but Docker works in the same way.</p><p>Initialize the container engine:</p><pre class="language-bash" data-lang="bash" v-pre=""><code class="lang-bash language-bash"><span class="token function">podman</span> machine init
<span class="token function">podman</span> machine start</code><button class="docsify-copy-code-button"><span class="label">Copy to clipboard</span><span aria-hidden="hidden" class="error">Error</span><span aria-hidden="hidden" class="success">Copied</span><span aria-live="polite"></span></button></pre><p>Common Operations. <span class="token function">podman</span> pull python:3.11-slim

<span class="token comment"># Run a container</span>
<span class="token function">podman</span> run <span class="token parameter variable">-it</span> python:3.11-slim

<span class="token comment"># List containers</span>
<span class="token function">podman</span> <span class="token function">ps</span> <span class="token parameter variable">-a</span>

<span class="token comment"># Stop container</span>
<span class="token function">podman</span> stop container_id

<span class="token comment"># Scan image for vulnerabilities</span>
<span class="token function">podman</span> scan myapp:latest

<span class="token comment"># Remove container</span>
<span class="token function">podman</span> <span class="token function">rm</span> container_id

<span class="token comment"># Remove all stopped containers</span>
<span class="token function">podman</span> container prune</code><button class="docsify-copy-code-button"><span class="label">Copy to clipboard</span><span aria-hidden="hidden" class="error">Error</span><span aria-hidden="hidden" class="success">Copied</span><span aria-live="polite"></span></button></pre><p>You can create a <code>Dockerfile</code> to build a container image.
